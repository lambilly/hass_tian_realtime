# 天聚数行-实时动态

通过天聚数行 API 获取实时数据，包括头条新闻、油价、汇率和空气质量信息，支持自动滚动显示。

## 主要功能

- 📰 实时热点新闻（支持自动滚动）
- ⛽ 各省市最新油价信息  
- 💵 实时美元兑人民币汇率
- ⛅ 指定城市空气质量指数
- 🔄 所有信息在滚动实体中循环显示

## 配置要求

- 天聚数行 API 密钥（免费申请）
- 选择油价查询省份
- 设置空气质量查询城市
- 配置数据更新间隔和滚动间隔

## 使用说明

1. 在天行数据官网注册并获取 API 密钥
2. 在 Home Assistant 中添加本集成
3. 填写配置信息
4. 集成将自动创建 5 个传感器实体
